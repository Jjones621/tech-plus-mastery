# CompTIA Tech+ Mastery Archive

A curated, free, public-resource archive for CompTIA Tech+ (FC0-U71).

## Start here

1. Download and read the official exam objectives.
2. Work through resources by domain.
3. Use practice tests to find weak areas.
4. Add notes and completed labs as you progress.

## Repository map

- `official/` — CompTIA pages and exam objectives
- `resources/` — curated courses, videos, notes, and practice tests
- `domains/` — study folders aligned to core Tech+ topics
- `labs/` — free hands-on exercises
- `tracking/` — progress and knowledge-gap templates

## Important

This repository links to publicly accessible material. It does not copy paid courses, exam dumps, or copyrighted question banks. Verify third-party resources independently and use official CompTIA objectives as the source of truth.
