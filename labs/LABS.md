# Free Hands-On Labs

## Hardware and operating systems

- Use Windows Settings and Task Manager to identify CPU, RAM, storage, and running processes.
- Create, rename, move, compress, and restore test files.
- Practice installing and removing a harmless application where permitted.

## Networking

- Run `ipconfig`, `ping`, `tracert`, and `nslookup` in Command Prompt.
- Identify your private IP, default gateway, and DNS server.
- Draw a simple home or office network diagram.

## Security

- Review browser privacy and permission settings.
- Turn on multifactor authentication for a personal test account.
- Compare a strong passphrase with a weak password.
- Identify phishing indicators in sample messages without opening links.

## Software development and data

- Create a small spreadsheet with fields, records, sorting, filtering, and validation.
- Write basic pseudocode using variables, conditions, and loops.
- Create a tiny GitHub repository and make several commits.

Only perform labs on systems and accounts you are authorized to use.
